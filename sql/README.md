# SQL Analysis

This folder contains SQL queries for the Banking Customer Churn & Profitability Analytics project.

The SQL queries focus on:
- Executive KPI reporting
- Customer segment profitability
- Churn risk analysis
- Product usage and churn
- Digital engagement and churn
- Support calls and churn
- Income band analysis
- Tenure band analysis
- State-level profitability
- High-value at-risk customer identification
- Low-product customer review
- Negative-profit customer review
- Product ownership analysis
- Cross-sell opportunity identification
- Retention priority reporting

These queries are designed to support banking analytics, customer retention strategy, profitability analysis, business reporting, and risk monitoring.
