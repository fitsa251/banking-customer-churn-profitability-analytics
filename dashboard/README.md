# Excel Dashboard

This folder contains the Excel executive dashboard for the Banking Customer Churn & Profitability Analytics project.

The dashboard summarizes:
- Total customers
- Total customer balance
- Total monthly profit
- Churn rate
- High churn risk customers
- Customer segment profitability
- Churn risk summary
- Product count and churn
- Digital engagement and churn
- Support calls and churn
- Income band profitability and churn
